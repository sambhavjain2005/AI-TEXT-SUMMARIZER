# AI-TEXT-SUMMARIZER
< this is a simple text summarizer code written in python
with the help of some ai tools >  
How It Works:
It uses the pipeline API from Hugging Face to create a summarization pipeline.

The model loaded is t5-small, a lightweight version of the T5 (Text-to-Text Transfer Transformer) model.

The user enters a paragraph via the command line.

The model processes the input and outputs a summarized version, adhering to length constraints.
